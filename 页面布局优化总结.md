## 响应式断点

| 屏幕宽度 | 设备类型 | 优化重点 |
|---------|---------|---------|
| < 992px | 平板横屏 | 菜单、间距 |
| < 768px | 平板竖屏 | 垂直布局、字体 |
| < 576px | 手机 | 全宽按钮、压缩间距 |
| < 360px | 小手机 | 最小化内边距 |

## 技术亮点

1. **CSS clamp() 函数**：实现流式字体大小
   ```css
   font-size: clamp(18px, 4vw, 24px);
   ```

2. **Flexbox 弹性布局**：灵活的响应式布局
   ```css
   display: flex;
   flex-direction: column;
   ```

3. **媒体查询**：针对不同屏幕的精确控制
   ```css
   @media (max-width: 768px) { }
   ```

4. **触摸优化**：提升移动端体验
   ```css
   -webkit-overflow-scrolling: touch;
   ```
